# Velocity Obstacles

Multi-agent collision avoidance simulation in C++ with Raylib.

## Prerequisites
- MSYS2 with `ucrt64` toolchain
- `g++` and `mingw32-make` in PATH

## Setup
```
mingw32-make setup
mingw32-make run
```

## Controls
- `R` Reset
- `V` Toggle VO cones
- `G` Toggle goal lines
- `A` Toggle velocity arrows
- `Click` Spawn agent
